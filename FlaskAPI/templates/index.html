Route Planning Map

To use the map, please select your origin and destination.
Your inputs will trigger an API call to Google Map's API, which will recommend one or more driving routes. We also collect realtime Chicago traffic congestion and weather data via API calls. These information will be fed to our predictive model, which will output the crash probability along each route.
Route color indicates safety, from green as safe to red as dangerous. We recommend the safest route with wider stroke.
